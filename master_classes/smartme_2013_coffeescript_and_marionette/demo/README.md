## How to use

* Install Node.js, npm and [Ruby](http://rvm.io/)
* For tests you need [PhantomJS](http://phantomjs.org/)
* install all deps
```bash
$ npm install
$ npm install -g bower
$ bower install # answer on all 2
```
* Run by `grunt serve`, build by `grunt` or `grunt build`
